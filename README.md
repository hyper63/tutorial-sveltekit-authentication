SvelteKit Authentication Demo Project

https://blog.hyper.io/sveltekit-authentication

## Developer Install Instructions

Prerequisites

* NodeJS v14+
* Git
* yarn - `npm install yarn -g`
* Github OAuth -- see blog post for more info
* .env file 
  VITE_CLIENT_ID
  VITE_CLIENT_SECRET

```
yarn
yarn dev
```
